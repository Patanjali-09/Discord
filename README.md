# Discord Clone

A responsive clone of Discord's landing page built using HTML and Tailwind CSS. The project focuses on recreating Discord's modern UI while ensuring responsiveness across different screen sizes.

## Features

* Responsive Design
* Modern Discord-Inspired UI
* Interactive Hover Effects
* Reusable Components

## Tech Stack

* HTML5
* Tailwind CSS
* Git & GitHub
